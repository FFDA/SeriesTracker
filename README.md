# SeriesTracker

Just an old project of mine. Allows to track TV series episode release dates, watched shows. Fetches data from IMDB using Imdbpie. Writer in Python3/Qt5

## Installation

Clone the repo and run series_tracker.py file

## Requirements

* Python3
* PyQt5
* ImdbPie (pip)
